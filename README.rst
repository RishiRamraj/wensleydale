Wensleydale
===========

Mr Wensleydale. Query Python, get the AST as JSON.

Support
-------

Currently only Python 3.5 is supported.

Usage
-----

.. code-block:: sh

  Usage: wensleydale [OPTIONS] PATH

    Mr Wensleydale. Query Python, get the AST as JSON.

  Options:
    --version  Show the version and exit.
    --help     Show this message and exit.
